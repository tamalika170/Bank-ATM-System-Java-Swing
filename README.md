## Bank ATM Management System (Java Swing)

An application-based Bank ATM Management System built using Java Swing and MySQL. Includes login, deposit, withdraw, mini statement, fast cash, and balance enquiry.


### Features
- User Login
- Signup Registration
- Deposit Money
- Withdraw Money
- Fast Cash
- Balance Enquiry
- Mini Statement
- Transaction History


### Technologies Used
- Java (Swing)
- MySQL Database
- JDBC
- Apache NetBeans

---

### Project Structure

- src/ → Java source files
- lib/ → External libraries (MySQL Connector, JCalendar)
- database/ → SQL database file
- src/img/ → Images used in project
- nbproject/ → May be ignored 

### How to Run the Project
1. Import project into Apache NetBeans
2. Add JAR files from lib folder to Libraries
3. Create MySQL database
4. Import SQL file from database folder
5. Run Login.java file
